# 英単語1級 — 4択クイズアプリ

英検1級レベルの英単語831語を、忘却曲線に沿って出題する4択クイズWebアプリです。
サーバー不要・完全無料でGitHub Pages上で動作します。

## ファイル構成

| ファイル | 役割 |
|---|---|
| index.html | アプリ本体（画面・ロジックすべて） |
| words.js | 単語データ831語（品詞タグ付き） |
| manifest.json / sw.js / icon-*.png | ホーム画面追加・オフライン対応用 |

## GitHub Pagesでの公開手順（無料）

1. [github.com](https://github.com) にログインし、右上の「+」→「New repository」
2. Repository name に `tango`（任意の名前でOK）と入力し、**Public** のまま「Create repository」
3. 作成後の画面で「uploading an existing file」リンクをクリック
4. このフォルダ内の **全ファイル**（index.html, words.js, manifest.json, sw.js, icon-192.png, icon-512.png）をドラッグ＆ドロップし、「Commit changes」
5. リポジトリの **Settings → Pages** を開き、「Branch」で `main` / `/(root)` を選び「Save」
6. 1〜2分後、`https://ユーザー名.github.io/tango/` でアプリが公開されます

## iPhoneでアプリのように使う

1. Safariで上記URLを開く
2. 共有ボタン（□↑）→「ホーム画面に追加」
3. ホーム画面のアイコンから起動すると全画面のアプリとして動作します（2回目以降はオフラインでも動作）

## 使い方

- 開いた瞬間に4択クイズが出題されます
- 右上のボタンで「EN→日本語」「日本語→EN」をいつでも切り替え可能
- 選択肢はすべて同じ品詞なので、品詞から正解を推測することはできません
- 正解すると復習間隔が延び（1分→10分→1日→3日→7日→16日→35日→90日）、間違えると最初に戻ります（忘却曲線に基づく間隔反復）
- 4回連続レベルに達した単語は「覚えた」扱いになります
- 「記録」タブで、日別の解答数・正解数・新たに覚えた数、間違えた単語・正解した単語・覚えた単語の一覧を確認できます
- 学習記録は端末のブラウザ内（localStorage）に自動保存されます

## 単語の追加・修正

words.js をGitHub上で直接編集できます（鉛筆アイコン）。1行1単語の形式です:

```js
{en:"英単語",ja:"日本語訳",pos:"v"},  // pos: v=動詞 n=名詞 adj=形容詞 adv=副詞
```

編集をコミットすると、数分後に自動で反映されます。
