# 英単語1級 — 4択クイズアプリ

英検1級以上・CEFR C1〜C2レベルの英単語を、忘却曲線に沿って出題する4択クイズWebアプリです。
サーバー不要・完全無料でGitHub Pages上で動作します。

## ファイル構成

| ファイル | 役割 |
|---|---|
| index.html | アプリ本体（画面・ロジックすべて） |
| words.js | 単語データ本体（831語収録。下記バッチ追加で最大2,582語） |
| words_add_batch1.js | 追加単語543語（words.jsに貼り付けて使用） |
| words_add_batch2.js | 追加単語691語（同上） |
| words_add_batch3.js | 追加単語517語・英検1級大問1対策（同上） |
| manifest.json / sw.js / icon-*.png | ホーム画面追加・オフライン対応・アイコン |

### 単語の追加方法

words.js をGitHub上で開き（鉛筆アイコンで編集）、最終行の `];` の**直前**に
words_add_batch1〜3.js の中身を丸ごと貼り付けてコミットしてください（コメント行ごとでOK）。
3つすべて貼り付けると合計 **2,582語** になります。
学習記録は単語の綴りをキーに端末内へ別途保存されるため、単語を追加しても記録は消えません。
既存単語の `en`（綴り）だけは変更しないでください（`ja` の訳語修正は自由です）。

## GitHub Pagesでの公開手順（無料）

1. [github.com](https://github.com) で「New repository」→ 名前は任意（例: `tango`）、**Public**で作成
2. 「uploading an existing file」から全ファイルをドラッグ＆ドロップして「Commit changes」
3. **Settings → Pages** で Branch に `main` / `/(root)` を選び「Save」
4. 1〜2分後、`https://ユーザー名.github.io/tango/` で公開されます

ファイルを更新したときは、該当ファイルをGitHub上で編集（または再アップロード）してコミットするだけで、数分後に反映されます。

## iPhoneでアプリのように使う

1. Safariで公開URLを開く
2. 共有ボタン（□↑）→「ホーム画面に追加」
3. ホーム画面のアイコンから起動すると全画面アプリとして動作します（2回目以降はオフラインでも動作）

※アイコンや画面を更新した場合、iPhoneのキャッシュにより反映が遅れることがあります。アイコンはホーム画面から一度削除して再追加すると更新されます（学習データは消えません）。

## 画面と使い方

**学習画面**（起動した瞬間に出題されます）

- 上部: 今日の解答数・正答率・覚えた単語数、右側に「学習/記録」タブ
- 単語カード（画面の約1/3）: 出題語、新規/復習バッジ、その単語の通算成績（正解◯回・ミス◯回）
- 下半分: 4択の選択肢。タップすると正解が緑・誤答が赤で表示されます
- 最下部の結果パネル: 左に正誤と品詞、右2/3の大きな「次へ」ボタンで即次の問題へ
- 右上のボタンで「EN→日本語」「日本語→EN」をいつでも切り替え可能
- 選択肢はすべて同じ品詞、かつ意味の重なる類義語は同時に並ばないため、消去法では解けません

**記録画面**（上部の「記録」タブ）

- サマリー: 覚えた単語・学習した単語・今日の解答数・連続学習日数
- 学習カレンダー: 月ごとのカレンダーに日別の解答数を表示（色が濃いほど多く学習）。日付タップで詳細（解答数・正解数・新たに覚えた数）、◀▶で過去の月に遡れます
- 単語一覧: 「間違えた」「正解した」「覚えた」の3タブで単語と成績を確認できます

## 出題の仕組み（忘却曲線）

正解するたびに次の出題間隔が延びます: 1分 → 10分 → 1日 → 3日 → 7日 → 16日 → 35日 → 90日。
間違えると最初の間隔に戻ります。4段階目（1日間隔を突破）に到達した単語が「覚えた」扱いです。
復習期限が来た単語を優先しつつ、新しい単語も混ぜて出題します。

## データの保存について

学習記録は端末のブラウザ内（localStorage）に自動保存されます。サーバーには何も送信されません。
端末やブラウザを変えると記録は引き継がれない点にご注意ください。
